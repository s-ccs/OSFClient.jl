OsfClient.osf_api — Method
osf_api(id;kwargs...)
osf_api(url;kwargs...)

Make requests against the osf api.

##args id/url: osf-id, e.g. 3a7kt or url containing "https://api.osf.io/v2/[...]". In the latter case, URL is not modified, in the former case, type and field need to be specified

kwargs

field: is put at v2/*type*/id/*field* type: is put at v2/*type* - defaults to nodes parse: if other than json returns the HTTP.get, else is converted to String and parsed with JSON.json

if parse == "json" (default), received data is put to string and parsed as json
